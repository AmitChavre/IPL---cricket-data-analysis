# IPL---cricket-data-analysis
# PROJECT OVERVIEW 
This project dives deep into 17 seasons of Indian Premier League (IPL) data — from the very first edition in 2008 through 2024. Using Python and data visualization, I transformed raw ball-by-ball data into meaningful insights about batting legends, bowling masters, toss strategies, winning patterns, and much more.
Whether you're a cricket fan, a data enthusiast, or a recruiter — this project shows how Exploratory Data Analysis (EDA) can turn millions of data points into clear, compelling stories.

# OBJECTIVES 
Identify the greatest run-scorers and wicket-takers in IPL history
Analyze how toss decisions influence match outcomes
Rank teams by all-time performance and win consistency
Break down dismissal patterns to understand how batters get out
Explore city-wise match distribution and home advantage
Uncover death-over bowling economy and pressure batting strike rates

## 📂 Dataset
 
| File | Description | Rows | Columns |
|------|-------------|------|---------|
| matches.csv | Match-level data — teams, toss, winner, venue | 1,095 | 20 |
| deliveries.csv | Ball-by-ball data — batter, bowler, runs, wickets | 260,920 | 17 |

### 🗓️ Dataset Coverage
 
```
Season Range  :  2008 → 2024  (17 seasons)
Total Matches :  1,095
Total Balls   :  260,920+
Teams Covered :  20+ franchises
Venues        :  30+ stadiums across India and abroad
```
 
> ⚠️ **Note:** The original Kaggle dataset covers seasons up to **2020**. Seasons 2021–2024 were sourced from extended community versions of the same dataset. For reproducibility, the Kaggle standard `matches.csv` + `deliveries.csv` format is used throughout.
 
📥 **Source:** [Kaggle — IPL Complete Dataset](https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020)
